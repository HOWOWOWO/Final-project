# 0050 ETF 投資分析網站 - Flask 專案說明文件

本專案為小組期末專題之最小可行性產品 (MVP)，使用 Flask 框架開發，並透過 Docker 進行環境管理，確保專案可以在不同電腦上快速重現。

## 專題主題說明
* **主題名稱**：0050 ETF 投資分析網站
* **簡單描述**：提供快速查詢股票的網站，以提供使用者快速了解各個股票的漲跌。

---

## 研究動機與背景
0050 ETF 是許多台灣投資人常見的長期投資選擇，因為它具有分散風險、投資標的透明與適合長期持有等特色。然而，一般投資人在查詢 0050 相關資訊時，常需要到不同網站分別查詢基本資料、成份股、歷年股利、現金殖利率、淨值與折溢價等資料，整理上較為不便。

因此，本專題希望建立一個 0050 ETF 投資分析網站，透過爬蟲收集相關資料，並使用 PostgreSQL 資料庫進行儲存，再利用 Flask 網站將資料呈現給使用者。透過表格、圖表與數學分析，協助使用者更清楚了解 0050 的投資價值與未來成長性。

題希望將程式設計、資料庫、網路爬蟲與資料分析結合，讓使用者不只是查詢資料，也能透過分析結果理解 0050 為何成為許多投資人的投資首選。

---

## 研究目的
本專題的研究目的，是希望建立一個以 0050 ETF 為主題的投資分析網站，透過網路爬蟲收集 0050 的相關資料，並將資料整理後儲存在 PostgreSQL 資料庫中，再利用 Flask 網站將資料以表格與圖表方式呈現給使用者。

本專題希望達成以下目的：

1.建立一個可以查詢 0050 ETF 資料的網站，讓使用者能快速了解 0050 的基本資訊、成份股、歷年股利、現金殖利率、淨值與折溢價等內容。<br>
2.透過爬蟲技術自動收集網站資料，減少人工查詢與整理資料的時間，提高資料整理的效率。<br>
3.使用 PostgreSQL 資料庫儲存資料，使資料能夠被有效管理、查詢與更新。<br>
4.結合數學與資料分析方法，觀察 0050 的股利變化、殖利率表現與折溢價情況，協助使用者判斷其長期投資價值。<br>
5.透過 Flask 與 Render 完成網站開發與部署，建立一個可以實際操作與展示的專題作品。<br>
6.藉由本專題訓練資料收集、資料庫設計、網站開發、資料分析與系統部署能力，讓作品未來能夠延伸成具備實用性的投資分析平台。
