# 0050 ETF 投資分析網站 - Flask 專案說明文件

本專案為小組期末專題之最小可行性產品 (MVP)，使用 Flask 框架開發，並透過 Docker 進行環境管理，確保專案可以在不同電腦上快速重現。

## 專題主題說明
* **主題名稱**：0050 ETF 投資分析網站
* **簡單描述**：提供快速查詢股票的網站，以提供使用者快速了解各個股票的漲跌。

---

## 專題動機
本專題以台灣 0050 ETF 作為研究對象，建立一個結合爬蟲、資料庫、網站開發與資料分析的投資分析平台。0050 是許多投資人長期投資台灣股市時常見的選擇，原因在於它追蹤台灣大型權值股，具有分散風險、操作簡單與適合長期持有等特性。

然而，對一般投資人而言，若只看股價高低，可能無法完整判斷 0050 的投資價值。因此本專題希望透過收集 0050 的基本資料、成份股、歷年股利、現金殖利率、淨值與折溢價等資料，進一步進行整理與分析，協助使用者觀察 0050 的歷史表現、配息狀況與未來成長性。

本專題不只是單純建立一個查詢網站，而是希望將數學與資料分析能力應用在金融投資情境中，讓使用者能用更直觀的方式理解 0050 為何會成為許多人投資台股的首選，也藉此展現 AI 與資料工具如何成為個人能力的放大器，讓作品未來有機會延伸成可落地的投資輔助平台。

---

## 專題目的
1.在 GitHub 建立專案 repo，進行版本控制與協同合作。<br>
2.使用 Flask 建立網站主程式。<br>
3.使用 Render 提供的 PostgreSQL 作為資料庫。<br>
4.使用爬蟲收集 0050 相關資料，包含動態網頁資料。<br>
5.將資料存入 PostgreSQL，並透過網站呈現。<br>
6.結合數學與資料分析，例如股利趨勢、殖利率分析、折溢價觀察。<br>
7.將網站部署到 Render，完成可公開展示的作品。<br>
8.製作 1080p、MP4 格式的專題介紹影片，說明動機、功能、程式碼與心得。<br>

## 環境開發工具
* **虛擬環境管理**：Docker / Docker Compose
* **後端框架**：Flask
* **前端樣板**：Jinja2 (HTML/CSS)
