0050 ETF 投資分析網站

研究動機與背景

0050 ETF 是台灣許多長期投資人的首選，具備分散風險與透明度高的特性。然而，一般投資人在查詢相關資訊時，常需要穿梭於各大財經網站分別抓取基本資料、成份股、歷年股利與折溢價。 本專題旨在打造一個整合平台，透過自動化技術將數據一網打盡，幫助使用者客觀評估長期投資價值。

專題研究目的

  • 自動化爬蟲減少人工整理時間
  • PostgreSQL 結構化高效數據管理
  • 結合 Chart.js 實現精準圖表視覺化
  • 提供穩健、配息等多元投資人類型分析

系統架構與資料流向

資料來源網站
Python / Selenium 爬蟲
資料清理與整理
Render PostgreSQL
Flask 後端系統
HTML / CSS / JS 前端

系統核心功能

基本與成分股查詢

完整揭露 0050 ETF 經理費率、規模，以及權重前 50 大成分股產業分布比例。

股利與殖利率分析

動態計算歷年配息數據與填息天數，利用圖表分析現金殖利率之長線吸引力。

淨值與折溢價追蹤

即時比對市價與基金淨值，協助投資人判斷當前市場是否過熱（溢價）或低估（折價）。

未來期望與擴充性

1. 多元標的擴充： 納入 0056、00878 等熱門高股息 ETF 進行橫向對比。

2. 自動化排程： 整合 GitHub Actions 或 Celery 實現數據每日定時全自動更新。

3. AI 投資理財助手： 預計加入大語言模型（LLM），根據歷史折溢價與殖利率數據生成自動化投資洞察報告。
